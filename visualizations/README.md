# Visualization Storyboards

This folder contains storyboard notes for future visualizations.

Potential animation formats:

- GIF,
- MP4,
- SVG animation,
- interactive notebook,
- or web-based visualization.

Suggested visualization assets:

| File | Purpose |
|---|---|
| `bmssp-flow.svg` | Lightweight animated overview used in the main README |
| `bounded-window.svg` | Shows how a boundary keeps part of the graph out of scope |
| `pivot-selection.svg` | Shows frontier compression into representative pivots |
| `relaxation-cycle.svg` | Shows an improved distance being classified into a queue interval |
| `recursive-slices.svg` | Shows how a parent call delegates bounded slices |
| `bmssp-recursion.gif` | Show recursive bounded calls |
| `pivot-selection.gif` | Show frontier compression into pivots |
| `bounded-queue.gif` | Show pull, insert, and batch prepend |
| `relaxation-flow.gif` | Show edge relaxation after a child call |

No benchmark or performance claim should be embedded in animations unless supported by reproducible code.
